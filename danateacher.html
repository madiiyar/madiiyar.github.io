HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure of the page content. Here are some basic HTML tags:

  • <html>
  • <head>
  • <title>
  • <body>
  • <h1>-<h6>
  • <p>
  • <ul>, <ol>, <li>
  • <a>
  • <img>

CSS Basics

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. Here are some basic CSS properties:

  • color
  • font-size
  • margin
  • padding
  • background-color
  • border
  • text-align
  • font-family
